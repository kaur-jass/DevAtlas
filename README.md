# 🚀 DevAtlas

> **Your Ultimate Developer Growth Platform**

DevAtlas is an all-in-one productivity platform designed for developers to track coding progress, organize learning, manage notes, analyze performance, and connect coding profiles from multiple platforms in one place.

Instead of switching between LeetCode, GitHub, Codeforces, notes, and bookmarks, DevAtlas provides a unified workspace to monitor your developer journey.

---

## ✨ Features

### 🔐 Authentication

* JWT Authentication
* Google Login
* Secure User Profiles

### 📊 Dashboard

* Developer analytics overview
* Coding statistics
* GitHub activity overview
* Study streak tracking
* Recent activity timeline

### 📚 Tracks

Dedicated workspaces for different learning paths:

* Data Structures & Algorithms
* Web Development
* AI / Machine Learning
* System Design
* Cyber Security

Each workspace is designed to become a personalized learning hub.

### 📝 Notes

* Create notes
* Organize notes
* Bookmark important concepts

### 🔍 Search *(Upcoming)*

* Global search powered by Elasticsearch
* Search across notes, problems, resources, and learning content

### 🤖 AI Mentor *(Upcoming)*

* Personalized learning recommendations
* AI-generated study plans
* Weak topic analysis

# ⚙️ Installation

```bash
git clone <repository-url>

```

### Backend

```bash
cd server
npm install
npm start
```

### Frontend

```bash
cd client
npm install
npm run dev
```
